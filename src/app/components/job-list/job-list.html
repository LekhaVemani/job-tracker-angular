🎯 Purpose of this Application

This Job Tracker application helps users manage and track their job applications efficiently.

  • 📌 Keep all job applications in one place
  • 📌 Track application status (Applied, Interview, Selected, Rejected)
  • 📌 Avoid forgetting applied companies
  • 📌 Stay organized during job search
  • 📌 Improve job tracking and decision making
📌 How to use this app:
  • 👉 Click Add Job to add a job
  • 👉 Fill Company, Role, Status
  • 👉 View jobs here
  • 👉 Use filter to sort jobs
  • 👉 Delete jobs if needed
